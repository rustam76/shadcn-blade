# Textarea

## Installation

In order to use the Textarea component, you need to add the component to your Laravel application using the following command:

```bash
php artisan shadcn:add textarea
```

## Usage

```html
<x-ui:::textarea placeholder="Type your message here." />
```

## Props

### Textarea

| Prop    | Description                  | Type     | Default |
| ------- | ---------------------------- | -------- | ------- |
| `class` | Additional classes to apply. | `string` | `''`    |
