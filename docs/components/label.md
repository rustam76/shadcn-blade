# Label

## Installation

In order to use the Label component, you need to add the component to your Laravel application using the following command:

```bash
php artisan shadcn:add label
```

## Usage

```html
<x-ui::label for="email">Your email address</x-ui::label>
```

## Props

### Label

| Prop    | Description                  | Type     | Default |
| ------- | ---------------------------- | -------- | ------- |
| `class` | Additional classes to apply. | `string` | `''`    |
