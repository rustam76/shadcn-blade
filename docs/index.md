# Shadcn Blade

Shadcn Blade is a collection of re-usable components built using Radix UI and Tailwind CSS, adapted for Laravel Blade.

## Introduction

This package allows you to easily integrate Shadcn UI components into your Laravel application using Blade components. It provides a CLI command to publish components to your project, giving you full control over the code.

## Documentation

- [Usage](usage.md)
- [Components](components.md)
- [Contribution](contribution.md)

## Installation

You can install the package via composer:

```bash
composer require rustamaji/shadcn-blade
```

## Requirements

- PHP 8.1 or higher
- Laravel 10.0, 11.0, or 12.0
